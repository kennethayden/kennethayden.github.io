In the mining industry, rock core samples are extracted to analyze the geological composition of a potential mine site and evaluate economic viability.
While the composition of these samples can be analyzed rapidly using highly sensitive laser-induced breakdown spectroscopy (LIBS) techniques, surface contaminants such as
dust and drilling debris appear as incorrect data in the analysis results. Traditional manual cleaning is labor-intensive, fails to match the high throughput of the scanning 
hardware, and introduces human variance in the cleaning quality, creating a severe operational bottleneck.

This project has been validated in the industry to deliver a standardized automated cleaning and accelerate the preparation of rock samples for analysis. The machine
removes human variance and guarantees a repeatable, high-throughput baseline for optimized spectroscopic data collection.
